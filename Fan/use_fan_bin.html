A review of Fan

added to Fann does not increase Fan's computability, since the backend will compile your language into Fann, it's essentially a syntactic sugar, no matter how complex the syntax sugar is, but bear in mind that every language is a syntax sugar of machine code.

Growing a language dynamically is a very very powerful feature, more importantly, it makes compiler building decentralized possible, for some features, the user does not need to submit a request and wait for several years, ending up a rejection. The main point is that building a compiler is not a rocket science, everyone could participate and submit their components, Fan helps to glue them together to provide the best user experience.
